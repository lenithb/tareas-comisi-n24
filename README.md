# Ejercicios de Práctica en JavaScript

Tres ejercicios básicos de JS en un solo archivo: `ejercicios.js`.

## Ejercicios

1. **Comparar números enteros** — pide dos enteros y muestra cuál es mayor (o si son iguales).
2. **Promedio de notas** — pide tres notas, calcula el promedio e indica si aprueba (≥ 6).
3. **Calculadora básica** — pide dos números y una operación (suma, resta, multiplicacion, division) y muestra el resultado.

## Cómo usar

Hay que vincular el archivo desde un HTML y abrilo en el navegador:

```html
<script src="ejercicios.js"></script>
```

Los datos se ingresan por `prompt` y los resultados se ven en la consola del navegador (`F12`).
