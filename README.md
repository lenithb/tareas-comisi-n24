# Ejercicios JS

Tres ejercicios de JavaScript para practicar Strings, Math y Switch. Incluye pequeña interfaz web bonita.

## Ejercicios

- **Strings** — ingresás una frase y una palabra, y el programa te dice si la encontró o no.
- **Math** — el programa genera un número secreto del 1 al 10 y tenés que adivinarlo.
- **Switch** — simulación de un restaurante: elegís un menú y te dice qué bebida te corresponde.

## Estructura

```
ejercicios/
├── index.html
├── styles.css
└── app.js
```

## Cómo usarlo

Clonar o descargar la carpeta y ejecutra /index.html.
