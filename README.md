# 📂 Ejercicios de JavaScript

Repositorio de ejercicios y prácticas de **JavaScript**, organizados en distintas ramas según el ejercicio. :D

---

## 🗂️ Estructura del repo

Cada rama contiene ejercicios específicos de los ejercicios. Para navegar entre ellas:

```bash
# Ver todas las ramas disponibles
git branch -a

# Cambiar a una rama específica
git checkout task/fecha
```

---

## Ramas

| Rama         | Contenido                           |
| ------------ | ----------------------------------- |
| `main`       | Información general del repositorio |
| `task/fecha` | Ejercicios por tema/unidad          |
| `task/fecha` | Ejercicios por tema/unidad          |
