# Ejercicios de Ciclos JS

Ejercicios de JavaScript para practicar ciclos `while` y `for`. Cada archivo es independiente y se puede probar directo en la consola del navegador o con Node.

## Archivos

**acumulador.js** — ciclo `while` que va sumando montos de ahorro diarios hasta llegar a $1000. Incluye validación para montos negativos.

**triangulos.js** — dos `for` anidados que imprimen un triángulo de asteriscos de 5 filas en consola.

**token.js** — simulador de autenticación con código fijo (5555) y máximo 3 intentos. Usa `while` con dos condiciones simultáneas.

**gestor_horas.js** — ciclo `for` que pide las horas de estudio de cada día de la semana, las acumula y al final muestra el promedio. Ignora valores negativos.

## Ejercicio 5 (pendiente)

**carrito.js** — simulador de carrito con descuentos progresivos. El `while` sigue agregando productos mientras el usuario confirme. Si el total supera $5000 se aplica un 10% de descuento.

## Cómo probarlos

Pegá el contenido de cada archivo en la consola del navegador, o ejecutalos con Node:

```bash
node acumulador.js
```

> Los ejercicios que usan `prompt` y `alert` deben correrse en el navegador.
